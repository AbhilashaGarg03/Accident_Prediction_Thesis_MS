# Accident Prediction Thesis

## Overview

A comprehensive master's thesis project focused on predicting traffic accidents using machine learning and data analytics techniques. This research explores the relationship between various environmental, temporal, and vehicular factors to develop predictive models for accident occurrence.

## Project Structure

- **Research Proposal**: Research_Proposal_Abhilasha.pdf - Initial research framework and methodology
- **Interim Report**: Abhilasha_Interim report.pdf - Progress report with preliminary findings
- **Final Submission**: Abhilasha_Thesis_Submission/ - Complete thesis submission materials
- **Analysis Notebooks**:
  - `Accident_Prediction.ipynb` - Exploratory data analysis and feature engineering
  - `ModelBuilding.ipynb` - Model development and evaluation
  - `merged_data.ipynb` - Data integration and preprocessing

## Research Objectives

- Identify key factors contributing to traffic accidents
- Develop predictive models using supervised learning techniques
- Evaluate model performance using appropriate metrics
- Provide actionable insights for accident prevention

## Dataset

Data sourced from multiple traffic and accident databases (see Data Dictionary for details). Additional datasets available at:

## Methodology

1. **Data Collection & Cleaning** - Aggregating and preprocessing traffic accident data
2. **Exploratory Data Analysis** - Statistical analysis and visualization of key variables
3. **Feature Engineering** - Creating meaningful features for model training
4. **Model Development** - Building and tuning multiple ML models (Logistic Regression, Random Forest, etc.)
5. **Model Evaluation** - Assessing model performance using cross-validation and test sets
